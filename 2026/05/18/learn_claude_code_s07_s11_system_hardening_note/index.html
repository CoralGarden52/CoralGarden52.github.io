九、一句话结论

Learn Claude Code s07–s11 的重点不是继续堆工具，而是给 agent 加上安全边界、扩展点、长期记忆、输入组装和恢复机制。

真正可用的 coding agent，不仅要能调用工具完成任务，还要能判断哪些动作能执行、哪些信息该保留、哪些输入该进入 prompt，以及失败后如何继续推进。
