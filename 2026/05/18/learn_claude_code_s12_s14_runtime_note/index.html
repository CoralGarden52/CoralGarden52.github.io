七、一句话结论

Learn Claude Code s12–s14 的核心不是复杂调度算法，而是把 agent 的工作拆成三层：任务目标层、运行执行层、时间触发层。

真正稳定的 coding agent，必须能区分“要做什么”“谁正在跑”“什么时候开始”，并让所有结果最终通过通知队列回到同一条主循环。
